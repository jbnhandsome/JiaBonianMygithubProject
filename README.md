# 创建基于GitHub的项目

1. 安装Git

1.1下载git

1.2安装git

2. 注册GitHub账号

3. 创建GitHub仓库(repository)

   仓库名字用“姓名仓库名”方式命名，使用驼峰式拼写

4. 创建公钥私钥

   4.1配置git账户

   4.2创建密钥

   4.3将公钥"id_rsa.pub"中的内容在GitHub中建立密钥

5. 将GitHub上建立的项目clone到本地磁盘

6. 使用同步脚本"syn.bat"

   6.1 既可以点击syn同步，也可以在idea中同步

7. Idea中进行git操作

   7.1更改完项目后，vcs commit

   7.2vcs push 成功提交到GitHub上更新项目
